## [Link to page](https://poliakh.github.io/myportfolio/site/itcreative_test/)  
Довольно емкое и очень интересно тестовое задание от одной компании, задачей было использовать чистый  css  без препроцессоров, БЄМ, флексы, выполнить адаптивную/отзывчивую верстку так же оптимизировать для IE11. Задание заняло у меня около 35 часов в целом.

`gulp build --prod` - сборка сжатой версии   
`gulp prod` -  отправка в папку продакшн, предварительно указать пусть к папке в   gulpfile.js
